# Chain Duel

Chain Duel is a Bitcoin and Lightning game experience built around real sats, open infrastructure, and global community participation.

This repository contains the public website for the project: [https://chainduel.net](https://chainduel.net).

## What Chain Duel is about

- Competitive gameplay with Bitcoin-native incentives
- Lightning-first payment flows and value distribution
- Open-source aligned ecosystem and contributors
- Community-driven growth through events, social channels, and supporters

## Why people follow Chain Duel

- **Play:** Join the game ecosystem and compete in sats-powered experiences
- **Learn:** Understand value splits and infrastructure choices in a simple visual format
- **Verify:** See real-world appearances and conference footprint
- **Support:** Contribute via funding, social reach, and open-source collaboration

## Website sections

- **Hero:** Project positioning and primary calls to action
- **Appearances:** Upcoming and past conference presence
- **Events Gallery:** Visual proof from global events
- **Videos:** Product and community footage
- **Splits:** Value-for-value contributor model
- **Infrastructure:** Bitcoin/Lightning open-source stack references
- **Contribute:** LNURL, GitHub, and community/social links
